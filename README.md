### 1. Pandas Core Data Structures

* **Importing Data:** Loading data from CSV and JSON files, including using external **Kaggle Datasets**.

* **Missing Values:** Identifying and handling `NaN` or null values.

* **Duplicates:** Finding and dropping redundant records.

* **Data Types:** Casting columns to appropriate types (e.g., strings to datetime).

* **String Manipulation:** Cleaning text data directly within DataFrame columns.


### 4. Selection, Filtering & Transformation

* **Indexing:** Selecting specific rows and columns efficiently.

* **Filtering:** Using conditional logic and the `.query()` method to extract specific subsets of data.

* **Transformation:** Applying custom functions to alter data.


### 5. Reshaping, Grouping & Combining

* **GroupBy & Aggregation:** Grouping data by categories to calculate metrics (Sum, Mean, Count).

* **Reshaping:** Using `Melt` and `Pivot` to transition data between wide and long formats.

* **Combining Data:** Merging, Joining, and Concatenating multiple DataFrames together (similar to SQL Joins).


### 6. Visualization

* **Basic Visualization:** Generating quick plots directly from Pandas to visually inspect data distributions and trends.
